# Arduino LED Sequence Project

This project demonstrates a simple LED sequencing system using Arduino. 
Three LEDs are connected to digital pins and light up one after another in a loop.

# Components Used
- Arduino Uno
- 3 LEDs
- Resistors
- Breadboard

# How It Works
Each LED turns ON and OFF with a delay, creating a running light effect.

# Skills Learned
- Basic Arduino programming
- Using digital output pins
- Understanding delays and timing
